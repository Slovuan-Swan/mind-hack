## Описание проекта

**«Игры разума» (Mind Games)** — это набор из пяти консольных игр, построенных по принципу популярных приложений для прокачки мозга. Каждый матч состоит из трех раундов. Игроку необходимо правильно ответить на все вопросы, чтобы победить. В случае одной ошибки игра завершается.

### Список игр:

1. **brain-even** — определение четного числа.
2. **brain-calc** — калькулятор (сложение, вычитание, умножение).
3. **brain-gcd** — нахождение наибольшего общего делителя двух чисел.
4. **brain-progression** — поиск пропущенного числа в арифметической прогрессии.
5. **brain-prime** — определение простого числа.

## Инструкция по установке и запуску

```bash
git clone https://github.com
cd mind-hack
npm install
npm link
```
